# ALVA Design System

AI-governed design system knowledge base.

## Structure

```
alva-lab/
├── design-bible.md          # Component architecture, layout philosophy, token hierarchy
├── ai-context.md            # AI agent instructions and hard constraints
└── .cursor/rules/           # Cursor IDE rules (token architecture, naming & states)
```

## Knowledge Files

- **`design-bible.md`** — canonical reference for component patterns, icon mask structure, composition rules, and token hierarchy.
- **`ai-context.md`** — constraints and conventions for AI-assisted code generation.
- **`.cursor/rules/`** — persistent Cursor rules for token architecture and naming conventions.

## Archived Experiments

Past implementations live in `_archive/` with datestamped folders.
